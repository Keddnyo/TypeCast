 Агрегатор дайджестов для 4PDA

## Features
1. Android 4.1+ / Windows 10+
2. Светлая и Amoled темы
3. Выбор из списка доступных разделов
4. Окно выбора дат
5. Копирование дайджеста по кнопке "Отправить"
