## Features
1. Android 4.1+ / Windows 10+
2. Light & Amoled themes
3. Forum navigation drawer
4. Auto remembering forum selection for next time app opening
5. Date range picker
6. Links highlighting
7. Clickable links
8. Digest copying while clicking "Send" button
