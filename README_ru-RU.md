## Features
1. Android 4.1+ / Windows 10+
2. Светлая и Amoled темы
3. Выбор из списка доступных разделов
4. Запоминание форума для следующего открытия
5. Окно выбора дат
6. Подсветка ссылок
7. Переход по ссылкам
8. Копирование дайджеста по кнопке "Отправить"
